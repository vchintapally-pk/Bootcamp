# Learn Scala

#### Rules of Thumb

 * SHOULD use immutable data structures
 * NEVER use "var"
 * No Loops! No Loopholes!!
 * Drop NULL from your vocabulary
 
Keep these rules in mind while writing scala code.

#### Recommended IDE
  * IntelliJ IDE - SBT Tool to build
